# Equipment data
## Data Input
Description | Enter the equipment description.  
---|---  
Shape factor | Shape factor to be considered for wind loading. Refer to the appropriate building code. If the equipment is close to the shell a value of 1.2 would be typical.  
Empty weight | Dry weight of the equipment.  
Operating weight | Operating weight of the equipment including operating liquid. NOTE : Must be greater than the _Empty weight_.  
Test weight | Weight of the equipment in the test condition. NOTE : Must be greater than the _Operating weight_.  
## Additional Information
Equipment can only be added to a vessel that has been specified as _Vertical_ in [Item orientation](<h9303.htm>) and has [Base Plate data](<h9120.htm>) defined in the model.
