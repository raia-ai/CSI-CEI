# Standard Flange Data

Standard flange dimensions are in accordance with _ASME B16.5_ [\[34\]](../h9998.htm#0034) for nominal diameters up to 24" and _ASME B16.47_ [\[33\]](../h9998.htm#0033) for larger diameters up to 60". ![](https://github.com/raia-ai/CSI-CEI/blob/img/stdflng.png)

| Specification    | Select the required specification, _ASME B16.5_ , _ASME B16.47 Series A_ or _ASME B16.47 Series B_.                                   |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| Nominal diameter | Select the nominal size.                                                                                                              |
| Rating           | Select the rating.                                                                                                                    |
| OK               | Select OK and the appropriate dimensions will be imported into the input. Dimensions, such as thickness, may be modified as required. |
