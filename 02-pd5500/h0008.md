# PD 5500 - Bolted or Welded Flat Heads
* * *
# Program Capabilities
This program performs calculations for unstayed circular flat heads and covers subject to internal pressure, in accordance with PD 5500 clause 3.5.5. If required, the calculations for the reinforcement of a single nozzle or opening in the flat head are performed in accordance with 3.5.5.3.1\(b\).
Flat heads or covers may be bolted, integrally forged, butt welded or fillet welded to the shell. The various types of flat head are specified from [Welded: Type 1](<h9991.htm#0001>) to [Welded: Type 3](<h9991.htm#0003>), [Welded: Type 8](<h9991.htm#0008>) and [Bolted: Type 1](<h9991.htm#0011>) to [Bolted: Type 4](<h9991.htm#0014>).
NOTE: The program does not include rectangular, elliptical or obround flat heads.
Bolting data for standard metric and inch series bolts is provided by the program. Clearances are based on TEMA TABLE D-5 - see [\[26\]](<h9998.htm#0026>). 
The deflection of bolted covers may also be checked, if necessary, in accordance with TEMA clause RCB-9.2.1 [\[26\]](<h9998.htm#0026>). 
Dependent on the [Bolt tightening method](<h9997.htm#0400>) selected, pre determined bolt loads can be applied. 
For bolted covers, the requirements of TEMA Section 5 RCB-11.7 [\[26\]](<h9998.htm#0026>) for [pass partition plates](<h9985.htm#1000>) can be considered, if required.

# Operation 
## Welded Flat Heads
Flat head types [Welded: Type 1](<h9991.htm#0001>) to [Welded: Type 3](<h9991.htm#0003>) are described in the following subsections. The corresponding identification letters from Figure 3.5-34 are shown. Weld details may be varied to suit the actual cover type shown in Annex E, Figure E.41.
The required thickness _e_ of welded flat heads is calculated in accordance with 3.5.5.3.1\(a\). For flat heads [Welded: Type 1](<h9991.htm#0001>) and [Welded: Type 2](<h9991.htm#0002>) the required cover thickness is calculated in accordance with Annex R.
The various limitations specified on Figure 3.5-34 for each head type are checked by the program.
### Integral Or Butt Welded
[Welded: Type 2](<h9991.htm#0002>) and [Welded: Type 3](<h9991.htm#0003>) are typically forged integral with the shell, or butt welded to the shell. These head types have an inside corner radius, and type [Welded: Type 3](<h9991.htm#0003>) has a straight flange section tapering down to the shell thickness.
### Fillet Welded To Inside Of Shell
[Welded: Type 1](<h9991.htm#0001>) is typically fitted inside the shell and attached by various weld types. See Annex E, Figure E.38 for recommended weld sizes, the default selection for the software is per figure E.38\(g\). 
[Welded: Type 8](<h9991.htm#0008>) caters for non standard weld types. 
## Bolted Flat Covers
Using the sketch identification letters from [Bolted: Type 1](<h9991.htm#0011>) to [Bolted: Type 4](<h9991.htm#0014>) the types of bolted flat heads included in the program are described in the following subsections. The corresponding identification letters from Figure 3.5-35 are shown.
### Edge Bending
[Bolted: Type 1](<h9991.htm#0011>) is a bolted flat head or cover with a gasket contact face that is entirely within the circle enclosed by the bolts.
The required thickness _e_ of the bolted flat head is calculated in accordance with the formulae given under sketch \(c\) on Figure 3.5-35 for the operating and bolting-up conditions, and is checked against the corroded thickness of the flat head. If a pass partition groove depth is specified which exceeds the internal corrosion allowance, then the actual thickness is measured to the bottom of the groove.
The minimum required edge thickness _e_ 1 at, or outside the gasket location is also calculated for the operating and bolting-up conditions.
### Facing Types
The various facing types available in the program for [Bolted: Type 1](<h9991.htm#0011>) which correspond to those commonly used for bolted flat heads are _Standard T.1a/b_ , _Tongue & Groove T.1c/d_, _Ring Type T.6_ and _O-ring_ \- see [Facing & Gasket Data](<h9985.htm>).
[Bolted: Type 2](<h9991.htm#0012>) and [Bolted: Type 3](<h9991.htm#0013>) are bolted flat covers with full-face gaskets extending outside the bolt circle.
The required thickness is calculated in accordance with 3.5.5.3.1, taking the diameter _D_ equal to the bolt circle diameter for [Bolted: Type 2](<h9991.htm#0012>), or equal to the opening diameter for [Bolted: Type 3](<h9991.htm#0013>). The required thickness is checked against the corroded thickness of the flat head. The cover type shown in the top sketch in Figure 3.5-35\(a\) can be specified as a [Bolted: Type 2](<h9991.htm#0012>).
_Standard_ or _Tongue & Groove T.1c/d_ facing types should be specified when using "flat" gasket materials. _Ring Type Joint_ or _O-ring_ facing types should be specified when using ring type joints or O-ring type gaskets respectively. For a _Tongue & Groove T.1c/d_ facing with a _raised_ face on the flat cover, the facing height is also deducted from the specified new thickness of the cover.
The surface finish on the flange face does not affect the calculations. However, reference should be made to Table 3.8-3 which gives recommended surface finishes for use with various gasket materials.
For a raised or flat face, a confined gasket may be specified if required - see [Facings with Internally Confined Gaskets](<h9985.htm#0250>)

### Self-Sealing Joint
[Bolted: Type 4](<h9991.htm#0014>) is a special type of cover with a self sealing joint and is normally used as a quick release manway cover. The required thickness _e_ of the bolted flat head is calculated in accordance with 3.5.5.3.1 using a value of factor _C_ determined from the formula given 3.5.5.3.1 a\) 1\) and is checked against the corroded thickness of the flat head.

## TEMA Requirements
If required, the deflection of bolted covers may be checked in accordance with TEMA clause RCB-9.2.1 [\[26\]](<h9998.htm#0026>). This requirement is intended to limit the deflection of bolted flat channel covers on heat exchangers to prevent loss of sealing between the gasket and the pass partition plate. For Bolted: Type 2 covers \(with full-face gaskets\) it is assumed that there is no edge bending due to the bolting so the second term inside the brackets in the equation for the deflection _Y_ is not considered.
The calculated deflection is checked against the TEMA recommended limits of 0.03 inches for diameters up to 24 inches, or 0.125% of the diameter for larger sizes.
## Nozzle Reinforcement
A single nozzle or opening in the flat head may be specified. An opening fitted with a nozzle must be either plain with no additional reinforcement or self reinforced as shown in [Plain nozzle](<h9993.htm#0501>) and [Self reinforced](<h9993.htm#0504>).
The nozzle to vessel attachment may be specified as _Set-on_ , _Set-in_ or _Set-through_.
The nozzle or opening may be centrally located or offset from the centre.
The thicknesses of the nozzle wall are checked against the requirements of 3.5.4.7\(a\) and 3.5.4.7\(b\).
The reinforcement of the nozzle or opening is calculated in accordance with 3.5.5.3.1\(b\).
## Test Conditions
Flat heads and covers may be checked for the shop and/or site hydrostatic test conditions by selecting the option _Check test case_.
For the test condition stresses are limited to 90% of the yield or material proof stress. Bolt stresses are limited to 150% of the design stress at room temperature.
If the design is not suitable for the test condition\(s\) appropriate warning messages will be produced. Detailed output data is not produced for the test conditions.

# System Messages
## Warning and Advisory Messages
Messages printed below in _italics_ are not considered to be critical - see [Warnings Messages](<h9997.htm#0501>).
_Acceptable calculation_ | The calculations have been performed successfully and the thickness of the flat head is sufficient for the design internal pressure.  
---|---  
_Bolt circle : max_ | The bolt circle diameter is too large to comply with the recommended bolt clearances. The value displayed shows the recommended maximum.  
_Bolt circle : min_ | The bolt circle diameter is too small to comply with the recommended bolt clearances. The value displayed shows the recommended minimum.  
Bolt stress | The bolt stress limit has been exceeded.  
_Bolt tensioning recommended_ | The nominal bolt diameter exceeds 38 mm - see PD 5500 3.8.1.4, or the _Probable wrench load_ predicted by ASME VIII, Div.1 Nonmandatory Appendix S is less than the actual bolt load. The use of bolt tensioning or torque tightening equipment is recommended.  
_Bolt tightening controlled_ | The bolt stress has exceeded the nominal code limit but may be acceptable for controlled bolt tightening if agreed between the designer, manufacturer and inspection authority. See also, ASME VIII, Div.1 Nonmandatory Appendix S.  
Code limitations exceeded | One or more of the limitations specified in PD 5500 for the shell have been exceeded. Use of the _Shells - Internal Pressure_ module will give detailed warning messages.  
_Contact width N\(min\)_ | The gasket width _N_ is less than the minimum from Table 3.8-4.  
_Design as flange_ | The diameter of the opening or nozzle exceeds _D_ / 2. It is recommended that the flat head should be designed as a flange - see 3.5.5.3.1\(c\).  
Dissimilar materials | The design strength of the nozzle _f_ b is outside the range 0.6 _f_ s to 1.5 _f_ s permitted by 3.4.5.3.7.  
Fillet weld leg | The leg length of the external fillet weld between the head and the shell is less than b/2 - see Fig. E38\(g\).  
Fillet weld leg: int | The leg length of the internal fillet weld between the head and the shell is less than b/2 - see Fig. E38\(g\).  
Insufficient bolting | The total required bolt cross-sectional area _A_ m is greater than the actual cross-sectional area _A_ b.  
Insufficient cover thk | The required thickness _e_ of the flat head or cover exceeds the actual thickness.  
_Insufficient pre-load_ | The specified bolt pre-load input is less than the design load required.  
_Length to weld_ | The required minimum length _w_ of the flat head's stub extends into weaker or thinner shell material.  
_Low temperature_ | Further analysis per Annex D may be required - see [Limitations](<h9996.htm#0700>).  
_Max nozzle thk_ | The actual thickness of the nozzle exceeds the Code limit. The program will discard excess thickness when calculating available reinforcement.  
_Min bolt dia_ | The nominal bolt diameter is less than the recommended - see 3.8.1.4. Also for small bolts the code recommends the use of torque spanners - see 3.8.1.4\(Note 1\).  
Min corner radius r | The corroded inside corner radius _r_ is less than the required minimum radius.  
Min nom thk to 3.5.4.7\(b\) | The nozzle neck thickness is less than the minimum specified in 3.5.4.7\(b\).  
Min reqd edge thk e1 | The minimum required corroded edge thickness of the cover at, or outside the gasket location is greater than the available thickness.  
Min tensile strength | The minimum tensile strength _R m_ specified exceeds the limit permitted for the construction category - see [Construction category](<h0098.htm#0600>)  
_Opening location_ | The opening or nozzle location should be checked to ensure that it does not foul adjacent components.  
_Reinf limit Lb_ | For self reinforcing types the reinforcing limit Lb based on Code rules include nozzle thickness that is unable to contribute to the reinforcement. A lower limit generally per ASME VIII Div.1 FIG. UG-40 \(e-1\) has been used.  
Shop test condition | Further calculations are required for the shop test condition.  
Site test condition | Further calculations are required for the site test condition.  
Slope of taper | The slope of the taper exceeds 1/4 \- see Figure 3.5-34\(a\).  
Temperature out of range | The design temperature exceeds the limit permitted for the construction category - see [Construction category](<h0098.htm#0600>)  
Thick shell theory reqd | The ratio of outside diameter to inside diameter, _D_ o __/ Di is greater than 1.3. The mean diameter rules are not necessarily applicable - see 3.2.2 NOTE 1.  
Thickness limit exceeded | The material reference thickness exceeds the limit permitted for the construction category - see [Construction category](<h0098.htm#0600>)  
Y > Ymax | The deflection _Y_ exceeds the recommended limit in TEMA clause RCB-9.2.1 [\[26\]](<h9998.htm#0026>).
