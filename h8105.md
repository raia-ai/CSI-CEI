# NozzlePRO FEA Analysis
Finglow now has the capability to run an FEA analysis on a nozzle or a saddle support using NozzlePRO.
The button will only be available when a calculation has successfully been performed using Finglow.
**NOTE:** The calculation cannot be performed without NozzlePRO being installed. See - NozzlePRO pathname
The calculation may take upto 5 minutes to run depending on its complexity, once selected the software may ask for any extra data that is not included in the Finglow calculation, this data must be input for the analysis to run correctly.
![](..\\img\\nozproext.png)
The software will display a progress bar to show that the calculation is running.
![](..\\img\\nozproprog.png)
Once the calculation has finished sucsessfully the output will be displayed using a webpage as shown below
![](..\\img\\nozproweb.png)
Followed by an application that displays all the relavent stress calculations and results.
![](..\\img\\nozprostress.png)
**NOTE:** The software has made some assumptions and will warn the user of these, please check the data carefully, for more options that are not covered by Finglow please run NozzlePRO using the file saved in the C:\temp\**Item name** \**calculation name** directory.
* * *
# NozzlePRO pathname
The pathname to the NozzlePRO installation must be entered, this can be done from the main menu by selecting **Tools- >NozzlePRO settings**, or you will be prompted before the software lauches NozzlePRO.
![](..\\img\\nozpropath.png)
Either type in the path manually of use the **Browse** button to browse your drives.
