# Editing design data
The software has the facility to create one or more design conditions for the vessel under consideration. By attaching these design conditions to a component the required calculations can be performed.
![](..\\img\\desdat2.png)
To edit a design condition, double click on the condition and the [Design Condition data](<h9103.htm>) screen will be displayed.
By right clicking on a design condition in the list a menu will be displayed with the following options available:-
**Add new condition** | This option will add a new design condition to the list.  
---|---  
**Edit design data** | This option allows the user to edit the design condition data with the [Design Condition data](<h9103.htm>) screen.  
**Delete condition** | This option will delete the currently selected design condition from the list.  
**Duplicate** | This option will create a duplicate of the currently selected design condition.  
When a new item is created it is given a default design condition.
