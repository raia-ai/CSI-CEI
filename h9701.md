# Data Sheet/Manual Calcs
* * *
# Program Capabilities
The Finglow Data Sheet and Engineering Formula Processor program is used to run pre-created Calculation or Data sheets. This option can be used in two different ways.
  1. It can be used to create completely user defined Data Sheets. These sheets can be set up by the user to reflect the standard layouts required for different items of process equipment.  
  2. The Engineering Formula Processor \(EFP\) provides data sheets with a highly comprehensive calculation capability enabling users to program their own Manual Calculation Sheets.
Both data sheets and calculation sheets can be printed along with any other Finglow Software output by using the Format Design Report option.
See also [Importing EFP Sheets](<h9708.htm>) and Convert SEF files.
* * *
# Operation
Operation of this module is the same as the other Engineering.
At this point the main difference between this and other modules is that no data will be loaded automatically into the computer's memory when the Data Sheet program is first accessed.
NOTE : All Systems are able to use this module in conjunction with any of the standard sheets issued by Finglow. However, the ability to create new Data and/or Calculation Sheets using the [EFP Editor](<h9703.htm>) is an optional extra.

## Convert SEF files
Files created prior to 2020 sofware release need to be converted to the new format. From the **File** menu select **Convert SEF files** , the dialog shown below will then be displayed.
![](..\\img\\FlatFileConvefp.png)
There are two different ways the software can display the users data - see [Folders or Drives view](<h9808.htm#0200>)
To find existing **SEF File\(s\)** navigate using either view. The files will be at the top level in the existing **DTA** folder. If the **DTA** folder contains any **SEF file\(s\)** the **Convert SEF files** button at the bottom of the screen will become live.
NOTE: This will only have to be done once.
