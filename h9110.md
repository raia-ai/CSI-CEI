# Stiffener data
## Data Input
Description | Enter the stiffener description.  
---|---  
Weld | Select _Full penetration, Continuous fillet_ or _Intermittent fillet_ to specify the stiffener to shell attachment weld. NOTE: If _Full penetration_ is selected then the lower flange is not input.  
Cold formed | Select _Yes_ if the stiffener is cold formed, or _No_ if the stiffener is hot formed or fabricated.  
External | Check this box to specify the stiffener as _External_.  
Stiffener Material | Shell material selection from the [Material Database](<h9996.htm>).  
## Additional Information
If there is no upper flange on the stiffener being specified, those fields should be zero.
## Error Messages
See also [General component error messages](<h9100.htm#1000>).
Invalid ring dimensions | One or more of the stiffener dimensions is invalid.  
---|---  
Stiffener weld | One or more of the stiffener weld dimensions is invalid.
