# Davit data
## Data Input
Description | Enter the davit description.  
---|---  
Weight | Weight of davit per unit length.  
Diameter | Effective diameter used in wind area calculation.  
## Additional Information
Davits can only be added to a vessel that has been specified as _Vertical_ in [Item orientation](<h9303.htm>) and has [Base Plate data](<h9120.htm>) defined in the model.
