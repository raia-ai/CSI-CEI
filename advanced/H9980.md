# Material properties

An alphabetic list of material properties, categorised by design code.

## PD 5500

| Material spec          | Material specification.                                                                                     |
| ---------------------- | ----------------------------------------------------------------------------------------------------------- |
| Allowable: ATM         | The design strength at room temperature - see [Design Strengths](../h0098.htm#0300).                        |
| Allowable: DT          | The design strength at the design temperature - see [Design Strengths](../h0098.htm#0300).                  |
| Elastic mod: ATM       | The elastic modulus (Young's modulus) at room temperature.                                                  |
| Elastic mod: DT        | The elastic modulus (Young's modulus) at the design temperature.                                            |
| Elastic mod E          | The value of elastic modulus used in the calculation.                                                       |
| Expansion coeff (xE6)  | The Expansion coefficient of the material at the mean metal temperature.                                    |
| Min tensile strength   | The minimum tensile strength _R m_ at room temperature - see [Minimum Tensile Strength](../h0098.htm#0400). |
| Design strength: ATM   | The design strength at room temperature - see [Design Strengths](../h0098.htm#0300).                        |
| Design strength: DT    | The design strength at the design temperature - see [Design Strengths](../h0098.htm#0300).                  |
| Design strength: Cat 3 | Design strength for a category 3 vessel - see [Construction category](../h0098.htm#0600).                   |
| Material SG            | The specific gravity of the material used in weight calculations.                                           |
| Poissons ratio         | Poisson's ratio _ν_.                                                                                        |
| Stress intensity       | The design strength at the design temperature.                                                              |
| Yield factor s         | The value of factor _s_.                                                                                    |
| Yield: ATM             | The specified minimum yield strength at room temperature _R e_ - see [Yield Strength](../h0098.htm#0400).   |
| Yield: DT              | The minimum yield strength at the design temperature _R eT_ - see [Yield Strength](../h0098.htm#0400).      |
| Reference thk          | The reference thickness used in the selection of material properties.                                       |
