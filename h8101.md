# Tool menu commands
* * *
## View 3D
From the Tools menu select **View 3D** or from the Toolbar click on the button ![](..\\img\\3ddraw.png)
Use this command to show or hide the 3d modelling window - see [View 3D](<h9304.htm#0000>).
![](..\\img\\3dwindow.png)
NOTE:For Nozzles with pads **View 3D** will represent the pad width as shown in the diagram below.
![](..\\img\\3d_nozpad.png)
* * *
## Toggle calc mode
From the Tools menu select **Toggle calc mode** or from the Toolbar click on the button ![](..\\img\\3dcalctog.png)
Use this command to toggle the calculation window ON or OFF.
* * *
## Item orientation
From the Tools menu select **Model orientation** or from the Toolbar click on the button ![](..\\img\\3dorient.png)
Allows the user to change the models orientation - see [Model orientation](<h9303.htm>).
* * *
## Default calculation data
From the Tools menu select **Default calculation data** or from the Toolbar click on the button ![](..\\img\\3ddefcalc.png) \- see [Default calculation data](<h9133.htm#0000>).
Allows the user to edit default calculation data.
* * *
## Design data
From the Tools menu select **Design data** or from the Toolbar click on the button ![](..\\img\\3ddesdat.png)
Allows the user to edit the relevant condition data for different Space/Voids in the vessel. Displays the [Editing design data](<h8800.htm#0000>) dialog box.
* * *
## Material manager
From the Tools menu select **Materials** then **Material manager**
Allows the user to change the materials for the entire model. see [Material manager](<h9306.htm#0000>)
* * *
## User defined materials
From the Tools menu select **Materials** then **User defined materials** or from the Toolbar click on the button ![](..\\img\\udeficon.png)
Alows the user to input material values for user defined materials - see [Editing user defined materials](<h9307.htm#0000>).
* * *
## Calculate selected
From the Tools menu select **Calculate selected** or from the Toolbar click on the button ![](..\\img\\3dcalcsel.png)
Performs the relevant calculations based on the design data entered, on only the currently selected items - see [How to calculate](<h8900.htm#0000>).
* * *
## Calculate all
From the Tools menu select **Calculate all** or from the Toolbar click on the button ![](..\\img\\calc.png)
Performs the relevant calculations based on the design data entered, on all the calculations for the entire vessel - see [How to calculate](<h8900.htm#0000>).
* * *
## Item Status
From the Tools menu select **Item Status** or from the Toolbar click on the button ![](..\\img\\3ditemstat.png)
This displays the [Item status](<h8900.htm#0001>) dialog box.
* * *
## Change component colour
From the Tools menu select **Component colour** or from the Toolbar click on the button ![](..\\img\\3dchngcolour.png)
This displays the [Change component colour](<h9301.htm#0001>) dialog box.
* * *
## EFP Datasheet
From the Tools menu select **EFP Datasheet**.
This displays the [Data Sheet/Manual Calcs](<h9701.htm>) dialog.
