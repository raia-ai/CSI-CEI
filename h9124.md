# Attachment data
## Data Input
Description | Enter the attachment description.  
---|---  
Type | Select the required attachment type as _Solid circular_ , _Rectangular_ or _Square_.  
Pad | Select the type of reinforcing pad. If no pad is fitted then select _None_.  
Pad Material | Pad material selection from the [Material Database](<h9996.htm>).  
## Additional Information
If [Load data](<h9123.htm>)![](..\\img\\3d_load.png) is inserted after an attachment the software will automatically produce a Local Load calculation.
## Error Messages
See also [General component error messages](<h9100.htm#1000>).
Offset | The offset has been specified outside the head.  
---|---  
Thickness | If a _Insert plate_ has been specified then its thickness must be greater than the shell thickness.  
Width | If a _Insert plate_ or _Reinforcing pad_ has been specified then its dimensions must be greater than the attachments.
