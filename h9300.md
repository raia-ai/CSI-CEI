# How to combine/copy nozzles
## Combine nozzles
Combine nozzles by right clicking on a nozzle in the calculation list and select **Combine** from the displayed menu, this will display the following screen.
![](..\\img\\combnoz.png)
This will give a list of possible nozzle combinations available to be combined with the currently selected nozzle. Select the required nozzle from the list and click the **OK** button. This will produce a combined nozzle calculation in the calculation list, and will grey out the the single calculation for each of the nozzles combined.
## Copy nozzles
By right clicking on a component in the component tree control and then selecting **Add to- >Nozzle** from the displayed menu, this will display the following screen.
![](..\\img\\copynoz.png)
This will give a list of possible nozzles available to be duplicated. Select the required nozzle from the list and click the **Copy** button. This will produce a copy of the selected nozzle and any components attached to the nozzle, clicking the **Cancel** button will cause a blank nozzle to be added.
