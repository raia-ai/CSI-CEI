# Basic commands
### Toolbar
The toolbar is displayed across the top of the application window, below the menu bar. The toolbar provides quick mouse access to many tools used in Finglow Software,
![](..\\img\\filereadcreate.png) | Load or Create an new item. Finglow Software displays the [Load/Create item](<h9801.htm#0700>) dialog box.  
---|---  
![](..\\img\\fileread.png) | Read an existing input file. Finglow Software displays the [Read Input](<h9801.htm#0100>) dialog box.  
![](..\\img\\filesave.png) | Save the currently entered data as an input file. Finglow Software displays the [Save Input](<h9801.htm#0200>) dialog box, in order to overwrite an existing input file or create a new one.  
![](..\\img\\fileout.png) | Save the complete calculation for later printing in report format using the Format Design Report option. Finglow Software displays the [Save Output](<h9801.htm#0300>) dialog box, in order to overwrite an existing output file or create a new one.  
![](..\\img\\fileprnt.png) | Send a copy of the current calculation only to the printer - see [Quick Print](<h9801.htm#0400>).  
![](..\\img\\drawpic.png) | Display a graphical representation of a component on the screen - see [Draw Component](<h8102.htm#0100>).  
![](..\\img\\calc.png) | Perform the relevant calculations based on the design data entered - see [Calculate](<h8102.htm#0200>).  
![](..\\img\\calcstat.png) | Display the calculation status box - see [Calculation status](<h8102.htm#0300>).  
![](..\\img\\unit.png) | Convert data from one set of units to the other - see [Unit Conversion](<h9997.htm#0101>).  
![](..\\img\\recfirst.png) | Move to the first column of data.  
![](..\\img\\recprev.png) | Move to the previous column of data.  
![](..\\img\\plus1.png) | Add a new column of data. \(Shortcut Ctrl+Insert\).  
![](..\\img\\minus1.png) | Delete the currently active column of data. \(Shortcut Ctrl+Delete\).  
![](..\\img\\recnext.png) | Move to the next column of data.  
![](..\\img\\reclast.png) | Move to the last column of data.  
![](..\\img\\hlpbtn.png) | Contextual Help \(Shortcut F1\).  
### Status Bar
![](..\\img\\statusb.png)
The status bar is displayed at the bottom of the Finglow Software window, it displays information about the current selections.
**Shells - Internal Pressure** | The currently selected Engineering Module. NOTE: This part of the status bar has Tooltip help which is shown above. The Tooltip gives more information about the module including its Version number.  
---|---  
**PD 5500** | The currently selected Design code.  
**Col 1 of 1** | For multi-column input, the current column number and the total number of columns.  
**HIM** | The User ID of the person currently logged on to the system.  
**\#1234** | The software serial number.  
**Field Type** | Indicates the type of input required. There are four types: _Numeric_ , _Text_ , _Menu_ and _Material Database_.  
**Max Columns** | Indicates the Maximum number of columns that can be input for the current input screen.
