# Basic commands

### Toolbar

The toolbar is displayed across the top of the application window, below the menu bar. The toolbar provides quick mouse access to many tools used in Finglow Software,

| ![](https://github.com/raia-ai/CSI-CEI/blob/img/fileread.png)     | Loads an existing Item or Creates a new Item - see [Load/Create Item](../h9800.htm#0300)                                                       |
| ----------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/wizbtn.png)       | Create a new Item using the [Vessel wizard](../h9800.htm#0700).                                                                                |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3dfilesaveas.png) | Allows the user to [Save as Item template](../h9800.htm#0400) for use when the current Item can be saved as a template for creating new items. |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/ExpJson.png)      | Allows the user to Export the model to a Json format file. See - [Importing & Exporting to JSON](../h9810.htm#0100)                            |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/ImpJson.png)      | Allows the user to Import a model from a Json format file. See - [Importing & Exporting to JSON](../h9810.htm#0100)                            |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/VesselLink.png)   | Exports the model to NozzlePRO Vessel Link. See - [NozzlePRO Vessel Link](../h9810.htm#0200).                                                  |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3ddraw.png)       | Allows the user to display the current Item as a 3D view.                                                                                      |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3dcalctog.png)    | Allows the user to switch between calculation and modelling modes.                                                                             |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3ddefcalc.png)    | Allows the user to edit default calculation data.                                                                                              |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3ddesdat.png)     | Allows the user to edit the relevant condition data for different Space/Voids in the vessel.                                                   |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/udeficon.png)     | Allows the user to edit the user defined materials database - see [Editing user defined materials](../h9307.htm#0000).                         |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3dcalcsel.png)    | Performs the relevant calculations based on the design data entered, on only the currently selected items.                                     |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/calc.png)         | Performs the relevant calculations based on the design data entered, on all the calculations for the entire vessel.                            |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3dorient.png)     | Allows the user to change the Item's orientation - see [Item orientation](../h9303.htm).                                                       |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3ditemstat.png)   | Displays the [Item status](../h8900.htm#0001) dialog box.                                                                                      |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/3dchngcolour.png) | Displays the [Change component colour](../h9301.htm#0001) dialog box.                                                                          |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/unit.png)         | Convert data from one set of units to the other - see [Unit Conversion](../h9997.htm#0101).                                                    |

### Status Bar

The status bar is displayed at the bottom of the Finglow Software window, it displays information about the current selections. ![](https://github.com/raia-ai/CSI-CEI/blob/img/3dstatusb.png) The status bar is displayed at the bottom of the Finglow Software window, it displays information about the current selections.

| **Version**           | The current version number of the software. NOTE: This part of the status bar has Tooltip help which is shown above. The Tooltip gives more information about the software version. |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **#1000**             | The software serial number.                                                                                                                                                         |
| **N mm degrees C**    | The currently selected set of units.                                                                                                                                                |
| **Software Status**   | The current status of the software.                                                                                                                                                 |
| **Weight and volume** | The approximate weight and volume of the entire vessel.                                                                                                                             |
| **Orientation**       | The current orientation selected for the vessel - See [Item orientation](../h9303.htm).                                                                                             |
