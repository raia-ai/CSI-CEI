## Searching the help

After clicking the ![](https://github.com/raia-ai/CSI-CEI/blob/img/f1_search.png) button from the toolbar the following dialog will be displayed:- ![](https://github.com/raia-ai/CSI-CEI/blob/img/searchdlg.png) This dialog allows the user to search the entire Finglow help for any word, by typing the required word into the edit box and pressing the **GO** button. The list box will then be filled with all the occurrences of that word. Double click on the required line to view that section of help. The last word entered into the Search box will appear in the edit box each time a new search is initiated.

***

## Contextual Help

To go directly to the help section for the screen currently displayed: press the **F1** key on the keyboard _or_ click on the ![](https://github.com/raia-ai/CSI-CEI/blob/img/hlpbtn.png) button Alternatively select _Contextual help (F1)_ from the Help menu: ![](https://github.com/raia-ai/CSI-CEI/blob/img/context_mnu.png).
