# Getting Started
## System Startup
On Startup the following dialog box is displayed
![](..\\img\\startup.png)
Read the information displayed and if acceptable press the _OK_ button and the System Log On dialog will be displayed. Press the _Cancel_ button to exit the software.
## System Log On
The following will be displayed
![](..\\img\\logon.png)
In the User edit box enter the user identification, in the Password edit box enter the password.
The default values are **HIM** and **TEST.** It is strongly recommended that these are changed after the first log-in - see [User Admin](<h9802.htm#0100>)
The User ID may be two or three characters, the password must be four characters. Please consult the System Administrator for an individual ID and initial password. The password is not displayed on the screen when it is typed in. If an invalid ID or password is entered, the software will display a message _Invalid entry_. Pressing the Cancel button or three invalid attempts will abort the log on procedure.
