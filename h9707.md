# EFP Status Bar
![](..\\img\\statusefp.png)
The status bar is displayed at the bottom of the Finglow Software window, it displays information about the current selections.
**Data Sheet Title** | The title of the currently selected Data Sheet. NOTE: This part of the status bar has Tooltip help which is shown above. The Tooltip gives more information about the module including its Version number.  
---|---  
**Field Type**  
| **Data Sheets** | Indicates the type of the selected edit box \(field\). There are three types of field: _Alpha Field_ , _Input Field_ and _Output Field_.  
| **EFP Editor** | This shows the Line and Column number of the cursors current position.  
**Link List** | The current link list. This is of the form _S:snum/E:enum/F:fnum_. _S_ , _E_ and _F_ refer to the Screen, Equation and Function files and _snum_ , _enum_ and _fnum_ are the corresponding index numbers for those files and denote which files have been linked together to produce the input file selected.
