# Channel-TS-Shell Connections
The following channel - tubesheet - shell connections are available. Note some Codes may have restrictions.
## Bolted-Bolted
Bolted channel and bolted shell. This configuration can either be simply supported between the gaskets or Edge Bolted.
![](..\\img\\blt-blt.png)
## Bolted-Welded
Bolted channel and welded shell. This configuration is always considered to be Edge Bolted.
![](..\\img\\blt-wld.png)
## Welded-Bolted
Welded channel and bolted shell. This configuration is always considered to be Edge Bolted.
![](..\\img\\wld-blt.png)
## Welded-Welded
Welded channel and welded shell.
![](..\\img\\wld-wld.png)

# Welded Tubesheet Connections
A welded tubesheet connection may be fillet welded or butt welded. If a raised stub is provided for a butt weld then the length of the raised portion standing proud of the tubesheet is required. 
![](..\\img\\TS_stub.png)
