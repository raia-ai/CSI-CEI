# Finglow Model 3D view
From the Toolbar press the ![](..\\img\\3ddraw.png) button, this will display the current item as a 3D model in a separate window.
Below is a list of actions that can be selected from the Toolbar displayed in this window.
**Button** | **Action**  
---|---  
![](..\\img\\updraw.png) | Move the model upwards.  
![](..\\img\\dndraw.png) | Move the model downwards.  
![](..\\img\\leftdraw.png) | Move the model to the left.  
![](..\\img\\rightdraw.png) | Move the model to the right.  
![](..\\img\\restoralldraw.png) | Restore the model to its original size and position.  
![](..\\img\\rotx1draw.png) | Rotate the model anti-clockwise about the X-axis. Alternatively move the mouse up while holding the left button down.  
![](..\\img\\rotx2draw.png) | Rotate the model clockwise about the X-axis. Alternatively move the mouse down while holding the left button down.  
![](..\\img\\roty1draw.png) | Rotate the model anti-clockwise about the Y-axis. Alternatively move the mouse right while holding the left button down.  
![](..\\img\\roty2draw.png) | Rotate the model clockwise about the Y-axis. Alternatively move the mouse right while holding the left button down.  
![](..\\img\\rotz1draw.png) | Rotate the model clockwise about the Z-axis.  
![](..\\img\\rotz2draw.png) | Rotate the model anti-clockwise about the Z-axis.  
![](..\\img\\restordraw.png) | Restore the model to its original position in the window.  
![](..\\img\\view1draw.png) | View the model along the X-axis in the positive direction.  
![](..\\img\\view2draw.png) | View the model along the X-axis in the negative direction.  
![](..\\img\\zoomindraw.png) | Zoom in on the model. Alternatively turn the mouse wheel upwards.  
![](..\\img\\zoomoutdraw.png) | Zoom out from the model. Alternatively turn the mouse wheel downwards.  
Mouse - right click | This will display the [Component Action](<h9000.htm#0001>) menu for the component under the cursor.  
Mouse - double click | This will display the component editing screen for the component under the cursor - see [Editing component data](<h9100.htm>).
