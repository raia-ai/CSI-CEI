# Skirt data
## Data Input
Description | Enter the skirt description.  
---|---  
Int FP thk | Thickness of any internal fireproofing.  
Ext FP thk | Thickness of any external fireproofing.  
Insulation thk | Thickness of any external insulation.  
Tolerance | Select option for input of thickness tolerance as either _Absolute_ or _Percentage_.  
Thinning | Thinning allowance to cater for mill tolerance on the skirt material. This tolerance will be deducted from the new thickness.  
Skirt Material | Skirt material selection from the [Material Database](<h9996.htm>).  
## Additional Information
A skirt can only be added to a vessel that has been specified as _Vertical_ in [Item orientation](<h9303.htm>).
The diameter of the skirt section is calculated by the software and is based on the [Skirt attachment data](<h9130.htm>) and the location of the [Circumferential weld](<h9113.htm>) adjacent to the skirt.
## Error Messages
See also [General component error messages](<h9100.htm#1000>).
Missing calculation data | An _Insulation thickness_ or _Fireproofing thickness_ has been specified but no weight data has been input. To input this data select the ![](..\\img\\3ddefcalc.png) button from the toolbar and then select _Vertical Vessels_ \- see [Default calculation data](<h9133.htm>)
