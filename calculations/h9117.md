# Floating head data

## Data Input

| Standard flange data       | Import [Standard Flange Data](../h9982.htm).                                                                                                                                                                                            |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Description                | Enter the floating head description.                                                                                                                                                                                                    |
| Facing                     | Select the required cover facing type - see [Facing & Gasket Data](../h9985.htm)                                                                                                                                                        |
| Facing configuration       | Select the required facing configuration as _Raised_ , _Flat_ or _Recessed_. NOTE: Not all facing configurations are available for each facing type.                                                                                    |
| Backing ring               | Select _Yes_ for a flange with a backing ring or _No_ for a flange with no backing ring. NOTE - If a _Full Faced_ facing type is selected then a backing ring cannot be specified - see [Backing Ring Flange Types](../h0011.htm#0230). |
| Keyed ring                 | Select _Yes_ for a keyed backing ring or _No_ for a Normal backing ring.                                                                                                                                                                |
| Split ring                 | Select the type of backing ring required.                                                                                                                                                                                               |
| Tolerance                  | Select option for input of thickness tolerance as either _Absolute_ or _Percentage_.                                                                                                                                                    |
| Thinning                   | Thinning allowance to cater for mill tolerance on cover material. This tolerance will be deducted from the new thickness.                                                                                                               |
| Bolt hole dia              | Enter the diameter of the bolt hole in the flange.                                                                                                                                                                                      |
| Quantity                   | Enter the quantity of bolts.                                                                                                                                                                                                            |
| Internally confined gasket | Check this box to specify an internally confined gasket.                                                                                                                                                                                |
| Cover Material             | Cover material selection from the [Material Database](../h9996.htm).                                                                                                                                                                    |
| Flange Material            | Flange material selection from the [Material Database](../h9996.htm).                                                                                                                                                                   |
| Backing ring Material      | Backing ring material selection from the [Material Database](../h9996.htm).                                                                                                                                                             |

## Additional Information

The cover offset is positive in the direction shown.

## Error Messages

See also [General component error messages](../h9100.htm#1000).

| Angle                    | The angle for a Keyed backing ring has been input greater than 80 degrees.                                                                                                               |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Backing ring thickness   | For a Keyed backing ring a Groove depth greater than or equal to the Backing ring thickness has been input - see [Backing Ring Flange Types](../h0011.htm#0230).                         |
| Confining/facing height  | The Facing has been defined as an _Internally confined gasket_ but no _confining height_ has been input, or no _facing height_ has been input                                            |
| Conflicting diameters    | The Tubesheet diameter doesn't match either the Backing ring or Keyed backing ring diameters - see [Backing Ring Flange Types](../h0011.htm#0230).                                       |
| Conflicting facings      | One or more of the facing dimensions doesn't match the other or is invalid, alternatively the facing configurations are incompatible - see [Facing & Gasket Data](../h9985.htm).         |
| Diameter                 | The Backing ring diameter is invalid.                                                                                                                                                    |
| Floating head dimensions | The Floating head flange diameter and/or thickness is invalid.                                                                                                                           |
| Height                   | The Facing has been defined as either _Raised_ or _Recessed_ but no _facing height_ has been input.                                                                                      |
| Offset                   | The _Offset_ measured from the centroid of the flange to the intersection of the flange ID and cover mean radius is invalid - see [Spherically Domed and Bolted End](../h0011.htm#0210). |
| Radius                   | An invalid Spherical radius has been input.                                                                                                                                              |
| Thickness                | An invalid Backing ring thickness has been input.                                                                                                                                        |
