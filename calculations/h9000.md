### Component Action

By selecting the required component and then clicking the right mouse button a pop-up menu will be displayed with the following entries. ![](https://github.com/raia-ai/CSI-CEI/blob/img/3dcompmenu.png)

| **Edit Data**                                                                                                                                                                               | Select this option to edit the component data see [Editing component data](../h9100.htm). NOTE: Double clicking the component will cause this option to be selected.                                                                           |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Drawing options**                                                                                                                                                                         | This menu allows the user to set drawing attributes for the component which determines how it will be displayed in the 3D drawing window - see Drawing options.                                                                                |
| **Insert before**                                                                                                                                                                           | Select this option to insert a new component before the current selection (i.e Above it in the tree control). This menu will expand to display a list of components, only the possible component combinations will be available for selection. |
| **Insert weld**                                                                                                                                                                             | This option will insert a circumferential weld in a cylinder.                                                                                                                                                                                  |
| **Insert after**                                                                                                                                                                            | Select this option to insert a new component after the current selection (i.e Below it in the tree control). This menu will expand to display a list of components, only the possible component combinations will be available for selection.  |
| **Add to**                                                                                                                                                                                  | Select this option to add a new branch component to the current selection. This menu will expand to display a list of components, only the possible component combinations will be available for selection.                                    |
| **Delete component**                                                                                                                                                                        | This will permanently delete the current selection and any componets that have been added to it from the model.                                                                                                                                |
| The user can also **Drag and Drop** components.                                                                                                                                             |                                                                                                                                                                                                                                                |
| When dragging a branch component (ie. Nozzles, Saddles, Attachments), either move the component from one place to another or make a copy of the component by holding down the **Ctrl** key. |                                                                                                                                                                                                                                                |
| When dragging an inline component (i.e. Welds, Shells, Flanges), a copy of the component will always be inserted inline after the component it was dropped on.                              |                                                                                                                                                                                                                                                |

### Drawing options

![](https://github.com/raia-ai/CSI-CEI/blob/img/3dcompmenudraw.png)

| **Reset**                    | This option will reset all drawing options for the entire model.                                                                                                                                     |
| ---------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Start here/Clear start**   | This option will set the selected component as the start point for drawing (i.e Nothing will be drawn before this component). If this component is already a start point the user can also clear it. |
| **Stop here/Clear stop**     | This option will set the selected component as the end point for drawing (i.e Nothing will be drawn after this component). If this component is already a stop point the user can also clear it.     |
| **Draw/Hide**                | This option will Hide or Draw the selected component.                                                                                                                                                |
| **Draw this component only** | This option will draw the selected component only.                                                                                                                                                   |
