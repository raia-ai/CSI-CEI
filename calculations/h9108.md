# Formed Head data

## Data Input

| Description        | Enter the head description.                                                                                                                                                                                                                                                                                                                                    |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Type               | Select the required head type - see [Shells & Heads](../h0098.htm#0100).                                                                                                                                                                                                                                                                                       |
| Diameter: OD-ID-NB | Select option for input of shell diameter as _Nominal bore_ , _External_ or _Internal_. For standard pipe sizes to ANSI B36.10 or BS 1600 select _Nominal bore_ - see [Nominal Pipe Sizes & Schedules](../h9986.htm). For other diameters and/or thicknesses select _External_ or _Internal_ - see [\[5\]](../h9998.htm#0005) and [\[11\]](../h9998.htm#0011). |
| Tolerance          | Select option for input of thickness tolerance as either _Absolute_ or _Percentage_.                                                                                                                                                                                                                                                                           |
| Thinning           | Thinning allowance to cater for mill tolerance on head material. This tolerance will be deducted from the new thickness.                                                                                                                                                                                                                                       |
| Insulation Thk     | Thickness of any external insulation.                                                                                                                                                                                                                                                                                                                          |
| Liquid height      | The height of any liquid above this head. NOTE : This data is only taken into account when performing an **External Loads on Vertical Vessels** calculation.                                                                                                                                                                                                   |
| Cold formed        | Check this box for cold spun heads.                                                                                                                                                                                                                                                                                                                            |
| Head Material      | Head material selection from the [Material Database](../h9996.htm).                                                                                                                                                                                                                                                                                            |
| Internal Space     | Head internal pressure space selection - see [Pressure Space](../h9103.htm).                                                                                                                                                                                                                                                                                   |

## Additional Information

The straight flange length is calculated by the software as the difference between the location of the heads tanline and the location of the adjacent weld.

## Error Messages

See also [General component error messages](../h9100.htm#1000).

| Height                   | An invalid head height has been input for a Ellipsoidal head.                                                                                                                                                                                                                          |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Missing calculation data | An _Insulation thickness_ has been specified but no weight data has been input. To input this data select the ![](https://github.com/raia-ai/CSI-CEI/blob/img/3ddefcalc.png) button from the toolbar and then select _Vertical Vessels_ - see [Default calculation data](../h9133.htm) |
| Radius                   | An invalid knuckle radius or spherical radius has been input for a Torispherical or Korbbogen head.                                                                                                                                                                                    |
