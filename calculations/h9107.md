# Cylinder data

## Data Input

| Description        | Enter the shell description.                                                                                                                                                                                                                                                                                                                                   |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Diameter: OD-ID-NB | Select option for input of shell diameter as _Nominal bore_ , _External_ or _Internal_. For standard pipe sizes to ANSI B36.10 or BS 1600 select _Nominal bore_ - see [Nominal Pipe Sizes & Schedules](../h9986.htm). For other diameters and/or thicknesses select _External_ or _Internal_ - see [\[5\]](../h9998.htm#0005) and [\[11\]](../h9998.htm#0011). |
| Tolerance          | Select option for input of thickness tolerance as either _Absolute_ or _Percentage_.                                                                                                                                                                                                                                                                           |
| Thinning           | Thinning allowance to cater for mill tolerance on shell material. This tolerance will be deducted from the new thickness.                                                                                                                                                                                                                                      |
| Insulation Thk     | Thickness of any external insulation.                                                                                                                                                                                                                                                                                                                          |
| Limpet coil        | Check this box if limpet coils are fitted to the shell.                                                                                                                                                                                                                                                                                                        |
| Shell Material     | Shell material selection from the [Material Database](../h9996.htm).                                                                                                                                                                                                                                                                                           |
| Internal Space     | Shell internal pressure space selection - see [Pressure Space](../h9103.htm).                                                                                                                                                                                                                                                                                  |

## Additional Information

The shell length is calculated by the software as the difference between the locations of the two adjacent welds. Also for information purposes the two adjacent components are shown. NOTE: If the model has been specified as _Vertical_ in [Item orientation](../h9303.htm) and has [Base Plate data](../h9120.htm) defined, then the maximum number of cylinders that can be added is limited to 10 - see [External Loads on Vertical Vessels](../h0012.htm#0305).

## Error Messages

See also [General component error messages](../h9100.htm#1000).

| ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Quantity                 | The maximum number of cylinders has been exceeded - see above.                                                                                                                                                                                                                         |
