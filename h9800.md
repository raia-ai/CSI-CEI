# Finglow 3D File commands
## Load/Create Item
For cloud users wanting to delete lock files. See [Delete lock files](<h9808.htm#0700>).
Item data stored using Finglow Software prior to the March 2020 release will need to be converted into the new file format. See [Convert item data](<h9808.htm#0100>).
From the File menu select **Load/Create Item** or from the Toolbar click on the button ![](..\\img\\fileread.png)
### Load an item
Before any calculations can be performed first _Load an item_ or _Create a new item_ if one does not exist.
To load the data for an _Item_ , navigate to the required folder select the _Item_ then click on the _Load_ button, or double click on the Item.
![](..\\img\\FlatFileRead.png)
### Create a new item
Navigate to the required folder and click the **New item** check box, then enter the **Item name** in the edit box and click the **New item** button.
![](..\\img\\FlatFileNew.png)
A new Item can also be created by using the **Duplicate** option - see [Rename, Delete or Duplicate](<h9808.htm#0400>).
NOTE: There are two different ways the software can display the users data - see [Folders or Drives view](<h9808.htm#0200>).
For a new model, the [**Item Templates**](<h9201.htm#0001>) will be displayed.
![](..\\img\\3d_newitem.png)
Select the template to be read: either double click on the template or press **Read** , to load the selected template or Item.
Set the appropriate orientation for the Item and click **OK**.
NOTE: The list shown is the standard list of templates that are provided with any new installation. They are assemblies of components with no input data.
User defined Item templates may be added to the list using the Save as Item template option.
## Vessel wizard
Finglow 3D has the option to create an Item using a Vessel wizard, this will create a new Item ready for calculation with just a few clicks.
From the File menu select **Vessel wizard**.
The following screen will then be displayed, navigate to the location for the destination Finglow file, then enter a new description into the edit box. Then click the **New item** button.
![](..\\img\\veswiz.png)
The next screen allows the input of the required dimensional data and also the design data to allow an Internal Pressure and Nozzle reinforcing calculation to be performed.
![](..\\img\\veswiz1.png)
The software will create a model that has 2 heads a shell and a nozzle with an attached flange.
**NOTE:** The Material selection will be applied to all components.
**NOTE:** The software will create a Horizontal Vessel with the datum on the right. This can be changed if required after the wizard has finished by clicking the button ![](..\\img\\3dorient.png) on the toolbar - see [Item orientation](<h9303.htm>).
Once the data has been input or the defaults accepted click the **OK** button and the following screen will be displayed, from here simply click the ![](..\\img\\calc.png) button from the toolbar and then the ![](..\\img\\calcprn.png) button to run all of the calculations and create output files ready for printing.
![](..\\img\\veswiz2.png)
## Save as Item template
From the File menu select **Save as Item template** or from the Toolbar click on the button ![](..\\img\\3dfilesaveas.png), this will pop-up the **Create Item** dialog box, which will allow the new item to be added to the list of Item templates.
![](..\\img\\3d_saveas.png)
Enter the description in the box provided and press the **Create Item** button, the Item component structure will be saved without any input data.
NOTE: This option is only available when logged on as the System Administrator.
## Import DC Model
From the File menu select **Import DC Model**
This option will import a model created using DesignCalcs into Finglow.
For more details on importing data see [Importing a DesignCalcs Model](<h9809.htm#0100>).
## Export Item to JSON
From the File menu select **Export Item to JSON**
This option will export a model to a JSON file format. **NOTE:** At present this file can only be read by Finglow.
## Import from JSON
From the File menu select **Import from JSON**
This option will import a model from a JSON file format. **NOTE:** At present this option can only read files created by Finglow.
## Convert item data
From the File menu select **Convert item data**
For more details on converting data see [Convert item data](<h9808.htm#0100>).
## Exit command
Use this command to end the Finglow session. From the File menu select **Exit** or use the Shortcut: Keys: _ALT+F4_.
