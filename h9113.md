# Circumferential Weld data
## Data Input
Description | Enter the weld description.  
---|---  
Type | Select weld type for the circumferential seam weld - see [Weld Types](<h0198.htm#0300>).  
Radiography | Select level of radiography for the circumferential weld - see [Radiography](<h0198.htm#0500>).  
## Additional Information
The location of a Circumferential weld is used by the software to determine the length of components, i.e the length of a cylindrical shell is determined by the location of the two adjacent circumferential welds.
## Error Messages
See also [General component error messages](<h9100.htm#1000>).
Component mismatch | Certain components cannot be connected together. For example: currently a _Cone_ cannot be directly connected to a _Formed head_ there has to be an intermediate _Cylinder_.
