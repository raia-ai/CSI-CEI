# Format Design Report
The Format Design Report menu offers the following commands:
Send Print Files to Printer | This option sends a complete design report to the printer or to a file for electronic storage or transfer via e-mail, it also allows the user to change the Client details and Revision history of a design report.  
---|---  
Reorder files | This option allows the user to Rename/Reorder/Delete print files.  
Print Setup | Select a printer and change the print settings.  
Output Language | Select the language for the printed output.  
* * *
# Send Print Files to Printer
NOTE : Before printing the calculation, it is advisable to use the Reorder files option to check the contents of the print output file and make any necessary changes or deletions.
![](..\\img\\okhist.png)
If the details are incorrect, press the _Edit data_ button to edit the revision status and client details as described in Edit Client and Revision History Data.
Press the _OK_ button to continue. After selecting the required printer the Print preview screen will be displayed, from here the entire document can be previewed before printing.
The printed output includes a Title page, an Index and a Revision History. Each page is numbered and includes the Item description, the Client's name, the Document number and Issue, the Calculation description, the Design Code, the Program title and version number, the Operator's initials and the Date when the calculation was performed.
**NOTE:** The print to Fabrication Manager option allows the user to export data to _Fabrication Manager_ \- see www.thinkcei.com
## Edit Client and Revision History Data
An important facility provided by the Finglow Software is its ability to maintain a revision history file for complete sets of calculations. This revision history is printed in front of the calculation data sheets. Each line of the revision history contains the issue number or letter, a description of the issue, the date and operator ID. This option is also used to enter any client details, this information is printed on the cover sheet of the printed report. The _Client_ and _Doc No._ are also printed at the top of each report page.
![](..\\img\\okhist2.png)
The client details input fields are all text fields and may consist of letters, numbers and other characters.
If the _Notes reqd ?_ option is set to **Yes** , explanatory notes for each type of calculation will printed at the start of each section.
Issue numbers are not generated automatically by the system and, prior to initial printout, the character ? in place of the issue number and the description _Preliminary issue_ are established by default. This enables a check print to be produced and any necessary corrections made before printing the first issue.
NOTE : The Issue number should be updated before the first official printout of the calculations is made.
Type in a single number or letter for the _Next Issue_ and then enter the _Description_ of the issue \(e.g. Issued for comments\). 
NOTE: For QA purposes, once entered it is not possible to edit the description of a previous issue number.
Press the _OK_ button to save the data entered, or the _Cancel_ button to abort without saving any changes.
The _Reset_ button will allow the user to delete the entire revison history and ,optionally, all existing print files.

# Reorder files
The print files for the selected Item will be displayed in groups relating to the Engineering Module and Design Code which created the particular print file, e.g. Shells - External Pressure, Nozzles & Openings, Bolted Flanged Connections, etc, as shown below.
![](..\\img\\edprndlg.png)
The descriptions entered by the user when the print file was saved are shown. To change the description: highlight the required print file, click the right mouse button and select _Rename_ from the Pop-Up menu and then type in the new description.
To delete any individual print file, highlight the required file and press the _Delete_ key on the keyboard, or click the right mouse button and select _Delete_ from the Pop-Up menu.
To change the order in which the calculations will be printed, highlight the file to be moved and drag and drop it on to the new position in the list, this will insert the calculation above the drop position. The calculation previously on that line and all subsequent calculations, will be moved down one line.
When finished, select the _Exit_ button.

# Print Setup
Use this command to select a printer and a printer connection, this is also used to changed the Font and margin settings for the output.
**Printer** | Select the printer to be used. Choose the Default Printer; or choose the Specific Printer option and select one of the currently available printers shown in the box.  
---|---  
**Properties** | Displays a dialog box where additional choices about printing can be made, specific to the type of printer selected.  
**Orientation** | Choose Portrait or Landscape.  
**Paper Size** | Select the size of paper that the document is to be printed on.  
**Paper Source** | Some printers offer multiple trays for different paper sources. Specify the tray here.  
**Network...** | Choose this button to connect to a network location, assigning it a new drive letter.  
After selecting **OK** on the Print Setup dialog box the _Change Font_ options are displayed.
## Change Font
This dialog allows the output font of three types of text to be changed, as shown below
![](..\\img\\chgfont.png)
Select the **Change Font** button next to the desired type of text to select a new font. After selecting the font style, the text in the appropriate window will be displayed using the new font. To accept the new font changes press the **OK** button, to exit without saving any changes select the **Cancel** button.
There is also an option to include a bitmap file, which will appear at the top of each page, replacing the standard company name text. To include a bitmap, first tick the **Include Bitmap** box and then enter a valid path to the bitmap file to be included. NOTE: That roughly 300 pixels of a bitmap is equal to 1 inch on the printout, so the bitmap file should be scaled accordingly.
After selecting **OK** on the Change Font dialog box the _Printer Margins_ are displayed.
## Printer Margins
This dialog allows the margin sizes to be changed, as shown below
![](..\\img\\margin.png)
The Top and Bottom margins are specified as a percentage of the paper length, and the Left and Right as a percentage of the paper width, of which they all have a maximum of 15%. To accept the new margin changes press the **OK** button, to exit without saving any changes select the **Cancel** button.

# Output Language
Finglow Software is supplied with two default languages - the first being that for data entry and system operation and the second for printout from the _Format Design Report_ option. Normally these two languages will be the same. If additional language modules have been purchased, using this option will enable the current set\(s\) of calculations to be printed in the selected language.
On selection of the _Output Language_ option a submenu of languages will be displayed on the screen as shown below. The currently selected language will be checked and the other available languages as per the licence configuration will be enabled.
