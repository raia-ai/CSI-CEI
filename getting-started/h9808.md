# File Management and Conversion

### Convert item data

Item data stored using Finglow Software prior to the March 2020 release will need to be converted into the new file format. From the **File** menu select **Convert item data** , the dialog shown below will then be displayed. ![](https://github.com/raia-ai/CSI-CEI/blob/img/FlatFileConvFolder.png) For information regarding converting EFP Sheets see - [Converting SEF files](../h9701.htm#0300) There are two different ways the software can display the users data - see Folders or Drives view Using either the Folders or Drives view navigate to the existing projects. Double clicking on a Project which will display a list of existing Items created using the **1902** or earlier version of the software, these Items can then be converted. The new converted file will be named _\[Project name]Item name_ and will be saved in the top level of the existing DTA folder. For example the file shown above with be saved as **C:\dta test\\\[Demo project]V-123 Vertical Vessel** and will have the extenstion **.F3D** for Items converted using Finglow 3D and **.F2D** for Items converted using Finglow 2D. When an Item has been converted a **Y** will be displayed next to that Item under the column header **OK**. Because the new file format is no longer an index based system the user is reminded that file descriptions now have to comply with normal system [File name conventions](../h9808.htm#0600). NOTE: All files assosiated with the Item will be converted, so this will only have to be done once for each Item.

### Folders or Drives view

#### Folders view

The **Folder(s)** view will show all the existing _dta_ folders that were set up previously using the **1902** or earlier versions, this data is read from the _ENV_ folder. ![](https://github.com/raia-ai/CSI-CEI/blob/img/FlatFileConvFolder.png)

#### Drives view

The **Drive(s)** view will show all the drives that are currently available on the computer. ![](https://github.com/raia-ai/CSI-CEI/blob/img/FlatFileConvDrive.png) NOTE: From either view the user can navigate backwards one level by clicking the ![](https://github.com/raia-ai/CSI-CEI/blob/img/navback.png) button at the top of the dialog.

### Link files

#### Exporting Link file

From the _File_ menu select _Export EFP file_ , this will then display the screen below. Click on the file to be exported. ![](https://github.com/raia-ai/CSI-CEI/blob/img/FlatFileLinkExp2.png) Once selected the location and description of the Export file may be specified. Once this is done the file can be Imported by another user. ![](https://github.com/raia-ai/CSI-CEI/blob/img/FlatFileLinkExp.png)

#### Importing Link file

From the _File_ menu select _Import EFP file_ , this will then display the screen below. Click on the file to be imported. ![](https://github.com/raia-ai/CSI-CEI/blob/img/FlatFileLinkImp.png) Once imported, the file will be copied to the Link file list and will be available for reading using the [Read new sheet](../h9701.htm#0400) option.

Rename, Delete or Duplicate For further details on creating input and output files - see [File commands: Finglow 3D](../h9800.htm) or [File commands: Finglow 2D](../h9801.htm) To delete Items or files, select the required item or file and press the _Delete_ key on the keyboard or click the right mouse button and select _Delete_ from the Pop-Up menu. Items or files may be modified by changing the description. To change the description highlight the required file, click the right mouse button, select _Rename_ from the Pop-Up menu and then type in the new description - see File name conventions. To duplicate Items, select the required item and click the right mouse button and select _Duplicate_ from the Pop-Up menu ![](https://github.com/raia-ai/CSI-CEI/blob/img/Filepop.png).

Delete lock files If the software ends unexpectedly then a lock file (\*.FNO) gets left behind in the directory where the currently loaded item resides, to delete this lock file(s) simply click the **Delete locks** button on the bottom of the screen below. ![](https://github.com/raia-ai/CSI-CEI/blob/img/FlatFileDelLok.png) **NOTE:** This option is only available for cloud users.
