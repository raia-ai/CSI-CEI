# Platform & ladder data
## Data Input
Description | Enter the platform & ladder description.  
---|---  
Platform | Select the type of platform. Either _Round_ for a round platform that extends 360 deg around the vessel, _Square_ for a square platform or _Segmental_ for a round platform where the user defines the extent.  
Caged ladder | Check this box to specify a caged ladder or un-check for a plain ladder.  
## Additional Information
Platforms & ladders can only be added to a vessel that has been specified as _Vertical_ in [Model orientation](<h9303.htm>) and has [Base Plate data](<h9120.htm>) defined in the model.
NOTE: Platforms are considered to be a fixed width specified by the software as 1000mm and have a fixed gap between the shell and platform of 200mm. Also the rail height is considered to be 1000mm.
## Error Messages
See also [General component error messages](<h9100.htm#1000>).
Missing calculation data | Weight data for the platform has not been input. To input this data select the ![](..\\img\\3ddefcalc.png) button from the toolbar and then select _Vertical Vessels_ \- see [Default calculation data](<h9133.htm>).
