# Blocking ring data
## Data Input
Description | Enter the blocking ring description.  
---|---  
Ring | Ring material selection from the [Material Database](<h9996.htm>).  
## Additional Information
Blocking rings can only be added to a simple vessel that has been specified as _Vertical_ in [Item orientation](<h9303.htm>) and has not got a [Tubesheet data](<h9116.htm>) or [Base Plate data](<h9120.htm>) defined in the model.
## Error Messages
See also [General component error messages](<h9100.htm#1000>).
Conflicting diameters | The _blocking ring OD_ must be less than the _jacket ID_.  
---|---  
Diameter | The _blocking ring ID_ must be greater than the adjacent _shell OD_.
