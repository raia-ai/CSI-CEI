# EN 13445 - Cyclic Loading
* * *
# Program Capabilities
This program performs calculations for cycling loading generally in accordance with EN 13445 Part 3 Sections 17 and 18.
NOTE: The calculations to include Thermal loads are not considered by the software.
The program can operate in two modes.
1\. Perform a simplified fatigue analysis based on pressure to Clause 17 
2\. Perform a fatigue analysis based optionally on pressure, stress or loading data to Clause 18.
* * *
# Operation
The data input represents the peak and trough values of a particular pressure, load or stress cycle as shown in the diagram below:
![](..\\img\\cyclic.png)
| _Case 1_ | _Case 2_ | _Case 3_  
---|---|---|---  
Stress | P1 | P2 | P3  
Stress | T1 | T2 | T3  
Reqd cycles | Q1 | Q2 | Q3  
The resultant stress ranges are assessed by the reservoir method as shown in the diagram below:
![](..\\img\\cyclic2.png)
Range | P2-T2 | P1-T1 | P3-T3  
---|---|---|---  
Reqd cycles | Q2 | Q1 | Q3  
## Material Categories
Currently EN 13445-3 clauses 17 and 18 are only applicable to the following material groups:
_Ferritic steel_
_Austenitic stainless steel_

# Explanation of Data
## Calculation Input
Please note that the following is a complete list of input data required for all types of cyclic calculations. The data is listed generally in the sequence in which it appears on the screen and in the printed output, but there may be some variations to the order depending on the options currently selected. The user will be required to enter only that data which is necessary for the particular type of calculation currently selected.
See [Data Input](<h9950.htm#0100>) for a full description of how to enter and edit data.
**DESIGN DATA**
Units | Select units - _N mm degrees C_ or _lbf inch degrees F_ \- see [Units](<h9997.htm#0100>).  
---|---  
Testing group | Select vessel testing group type - See [Testing Groups](<h0298.htm#0500>).  
Simplified Analysis | Select _Yes_ to evaluate if a fatigue assessment is required and carry out a simplified anaylsis to Clause 17 if necessary or select _No_ to run a full fatigue analysis to Clause 18 .  
Bolted | Select _Yes_ for bolting material or _No_ for plate/forgings, etc.  
Weld type | Select weld type for the longitudinal seam weld - see [Welded Joints](<h0298.htm#0400>).  
Weld classification | Select from the list of weld classifications as per Table 17-4.  
Class A | Select _Yes_ for fatigue efficient equipment that comply with the requirements of 17.5.4.2.  
Notched | Select _Yes_ to account for the Notch effect as per 17.6.2.3.  
Radius r | The notch radius r as per Figure 17-3.  
Surface condition | Select surface condition type from the list as per Table 18-8.  
Roughness | If _User defined_ has been selected from the _Surface condition_ menu then enter the _Roughness R z_ as per 18.11.1.1.  
Material spec | Material selection from the [Material Database](<h9996.htm>).  
Material | Material specification or description. If not using the [Material Database](<h9996.htm>) option the following data will be required.  
Min tensile strength | The minimum tensile strength _R m_ for the material at room temperature - see [Yield strength and Tensile strength](<h0298.htm#0800>).  
Yield: ATM | The specified minimum yield strength _R p 0.2_ for the material at room temperature - see [Yield strength and Tensile strength](<h0298.htm#0800>).  
Yield: ATM \(1.0%\) | The specified minimum yield strength _R p 1.0_ for the material at room temperature - see [Yield strength and Tensile strength](<h0298.htm#0800>).  
Min tensile strength: DT | The minimum tensile strength _R_ m/T for the material at the design temperature - see [Yield strength and Tensile strength](<h0298.htm#0800>).  
Yield: DT | The yield strength _R p 0.2/T_ for the material at the design temperature - see [Yield strength and Tensile strength](<h0298.htm#0800>).  
Yield: DT \(1%\) | The yield strength _R p 1.0/T_ for the material at the design temperature - see [Yield strength and Tensile strength](<h0298.htm#0800>).  
Elastic modulus E | The Elastic modulus E if a user defined material has been specified.  
Design temp | Design temperature.  
**DESIGN DATA**
Calculate stresses | Select _Yes_ to calculate stresses from a given bolt load or shell internal pressure.  
---|---  
Cylindrical shell | Select _Yes_ to calculate stress for a cylindrical or _No_ for spherical shell portion.  
Calc nozzle SCF | Select _Yes_ in order to calculate stress concentration factor for nozzles in accordance with PD 5500 Annex W.6.2.2.3.  
Diameter: OD-ID-NB | Select option for input of diameter as _Nominal bore_ , _External_ or _Internal_. For standard pipe sizes to ANSI B36.10 or BS 1600 select _Nominal bore_. For other diameters and/or thicknesses select _External_ or _Internal_ \- see [\[5\]](<h9998.htm#0005>) and [\[11\]](<h9998.htm#0011>).  
Nominal bore | Select the required nominal bore - see [Nominal Pipe Sizes & Schedules](<h9986.htm>).  
Schedule | Select the required pipe schedule - see [Nominal Pipe Sizes & Schedules](<h9986.htm>).  
Inside diameter | Inside diameter of a cylindrical or spherical shell.  
Outside diameter | Outside diameter of a cylindrical or spherical shell.  
New thickness | New or as built thickness of a shell. Also required when _Calculate stresses_ is set to _No_ , in order to obtain material properties for those materials with thickness dependencies.  
Nominal diameter | Bolt nominal diameter required when _Calculate stresses_ is set to _No_ , in order to obtain material properties for those materials with thickness dependencies.  
Tolerance | Select option for input of thickness tolerance as _Absolute_ or _Percentage_.  
Thinning | Thinning allowance to cater for mill tolerance on shell material. This tolerance will be deducted from the new thickness.  
Corrosion: int | Internal corrosion allowance to be deducted from the as built shell thickness.  
Corrosion: ext | External corrosion allowance to be deducted from the as built shell thickness.  
Bolt area Ab | For bolted materials, the total area for bolting.  
Pressure Pmax | Maximum allowable pressure _P max_ permitted for the component, calculated from the appropriate module.  
Factor eta | Factor _η_ from Table 17-1.  
**NOZZLE DATA**
Opening ident | Nozzle or opening identification number or name.  
---|---  
Diameter: OD-ID-NB | Select option for input of diameter as _Nominal bore_ , _External_ or _Internal_. For standard pipe sizes to ANSI B36.10 or BS 1600 select _Nominal bore_. For other diameters and/or thicknesses select _External_ or _Internal_ \- see [\[5\]](<h9998.htm#0005>) and [\[11\]](<h9998.htm#0011>).  
Nominal bore | Select the required nominal bore - see [Nominal Pipe Sizes & Schedules](<h9986.htm>).  
Schedule | Select the required pipe schedule - see [Nominal Pipe Sizes & Schedules](<h9986.htm>).  
Nozzle OD | Outside diameter of the nozzle neck. NOTE: For a [Self reinforced](<h9993.htm#0504>) nozzle type this should be the diameter at the shell.  
Nozzle ID | Inside diameter of the nozzle neck.  
Nozzle thk | New or as built thickness of the nozzle neck.  
Tolerance | Select option for input of nozzle wall thickness tolerance as _Absolute_ or _Percentage_.  
Thinning | Thinning allowance to cater for mill tolerance on the nozzle wall thickness, typically 12.5% for pipe material. NOTE: For _Self reinforced_ and _Butt welded insert_ nozzle types the thinning allowance may only be input in mm or inches, not as a percentage.  
**CYCLIC LOADING DATA**
A maximum of 10 columns of data may be input.

---|---

Pressure | The peak and trough values of internal pressure.  
Stress | The peak and trough values of stress.  
Reqd cycles n | The total number of cycles required for this case.

# System Messages
## Warning and Advisory Messages
Messages printed below in _italics_ are not considered to be critical - see [Warnings Messages](<h9997.htm#0501>).
_Acceptable calculation_ | The calculations have been performed successfully and the combined usage factor is less than the limit.  
---|---  
Code limitations exceeded | The limitation for the maximum pressure cycle range has been exceeded - see 17.5.4.2.  
Damage factor | The total damage factor exceeds 1.0.  
_Elastic plastic_ | The effects of plasticity have been taken into account - see 17.6.1.4.  
EN 13445-5 Table 6.6.1-1 | One or more of the parameters in the table Testing groups for steel pressure vessels is outside the limits.  
Extrapolated graphs/tables | The limits of Table 18-10 has been exceeded.  
Factor Kt | A value of factor _K t_ has been assumed.  
_Low temperature_ | Further analysis per EN 13445-2 Annex B may be required - see [Limitations](<h9996.htm#0700>).  
Pressure scf | The parameter limits in the nozzle stress calculation have been exceeded.  
Temperature out of range | The limits in 17.4.4 have been exceeded. The maximum value will be displayed.  
## Fatal Warning Messages
See [Fatal Messages](<h9997.htm#0502>).
Dia and/or thk invalid | A diameter and/or thickness is zero or the thickness exceeds the radius.  
---|---  
Input | An item of input data has not been entered.  
Material | There has been an error getting data from the [Material Database](<h9996.htm>), a more detailed explanation of this error will displayed on the Material Dialog box of the specific material. Or a Material has been selected that does not comply with the rules in 17.4.3.  
Method failure | Component dimensions are such that the calculation method has failed - see also [Fatal Messages](<h9997.htm#0502>).  
Min tensile strength | When performing the Elastic plastic check, the Minimum tensile strength has exceeded the maximum - see 18.8.1.1.  
Permitted Cycles | One or more of the permitted cycles calculated is zero.  
Thickness | A new or as built thickness is less than the sum of corrosion/tolerance allowances.  
Tolerance | The thickness tolerance is greater than the new or as built thickness.  
Weld classification | A weld classification has been selected that does not comply with the restrictions in 17.5.3.
