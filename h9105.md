# Longitudinal Weld data
## Data Input
Description | Enter the weld description.  
---|---  
Type | Select _Yes_ for a petal type configuration, or _No_ for an offset type configuration. NOTE: This option is only displayed for welds attached to a Formed Head.  
Quantity | The quantity of petals the Formed Head is split up into.  
Type | Select weld type for the circumferential seam weld - see [Weld Types](<h0198.htm#0300>).  
Radiography | Select level of radiography for the circumferential weld - see [Radiography](<h0198.htm#0500>).  
## Additional Information
None.
## Error Messages
See also [General component error messages](<h9100.htm#1000>).
Diameter | If the _Type_ menu has been set to _Yes_ then the diameter input must be less than 80% of the Formed head diameter.  
---|---  
Length | If the _Type_ menu has been set to _No_ then the offset input must be less than 30% of the Formed head diameter, or 25% of the Flat cover diameter.
