# Additional loads data
If the load data is attached to a shell, head or cone then the load is considered to be an additional load, and would be included as such in a calculation for either _Vertical Vessels_ or _Horizontal Vessels_.
If the model has been specified as _Vertical_ in [Item orientation](<h9303.htm>) then see Additional loads on a vertical vessel otherwise see Additional loads on a horizontal vessel.
## Additional loads on a vertical vessel.
Description | Enter the load description.  
---|---  
Mass | Weight or mass of attachment.  
Wind area | Bluff area exposed to wind. NOTE: only considered for Erection, Operating and Test cases.  
Shear | Any applied shear force. NOTE: only considered for Erection, Operating and Test cases.  
Moment | Any applied external eccentric moment. NOTE: not considered for Lifting cases.  
## Additional Information
None.
* * *
## Additional loads on a horizontal vessel.
Description | Enter the load description.  
---|---  
Mass | Additional point mass.  
Load X \(shear\) | Load in the longitudinal direction.  
Load Y \(shear\) | Load in the vertical direction.  
Load Z \(shear\) | Load in the transverse or lateral direction.  
Moment Y | Moment about the Y axis.  
Moment Z | Moment about the Z axis.  
## Additional Information
None.
