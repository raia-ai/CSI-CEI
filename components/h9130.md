# Skirt attachment data

## Data Input

| Description   | Enter the skirt attachment description.                                                                                                                                                                                                                                                    |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Fillet welded | Check this box if the skirt is _Fillet welded_ to the side of the vessel, or un-check if not. If the skirt is _Fillet welded_ then the location of the following skirt section is input by the user, otherwise the software will calculate the skirt location based on the diameter input. |
| Flared skirt  | Check this box if the skirt is Flared, or un-check if it is a straight skirt.                                                                                                                                                                                                              |

## Additional Information

None.
