# Default calculation data

| Allow surface factor   | Multiplication factor to be applied to the _Allowable stress: DT_ to obtain the allowable surface stress in the shell. The recommended value is 3.0 - see [Shell Stresses](../h0198.htm#1200).                                                  |

| Check test case             | Check this box to perform a test case assessment of the Shop test, and optionally the Site test, conditions - See [Test Pressures](../h0098.htm#0200).                                                 |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |

| Calculation method          | Select _Standard_ to perform calculations to PD 5500, or _EN 1591_ to use the EN 1591 method.                                                                                                          |
| Applied axial tensile force | External axial force applied to the shell to which the flange is attached. NOTE: A tensile force should be entered as a positive value and a compressive forces should be entered as a negative value. |
| Applied lateral force       | For the EN1591 method, External resultant lateral force applied to the shell to which the flange is attached - see 7.2.2.2.                                                                            |
| Applied bending moment      | External bending moment applied to the shell, acting about an axis normal to the axis of the shell.                                                                                                    |
| Applied torsional moment    | For the EN1591 method, External torsional moment applied to the shell to which the flange is attached - see 7.2.2.2.                                                                                   |

#### EN 1591 - Calculation Method

| Quantity NR          | The number of re-assemblies and re-tightenings during service life of joint.                                                                                                                                                                                     |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Friction coefficient | The friction coefficient for the bolts. Values from 0.1 (smooth, lubricated surfaces) to 0.35 (rough, dry surfaces) are permitted.                                                                                                                               |
| Friction coefficient | The coefficient _μ G_ which can be assumed between the gasket and flange. Examples of values are 0.05 (PTFE based gaskets) or 0.15 (Flat metalic gaskets) - see Table E.1. NOTE: This value may be zero if no Applied loads/moments have been input - see 7.4.2. |
| Factor cB            | Bolt correction factor _c B_ cannot be greater than 1.0 - see BS EN 1591-1 Clause 7.2.                                                                                                                                                                           |
| Leak rate            | Select the specific gasket leakage rate from the menu.                                                                                                                                                                                                           |
| Pressure Qmin        | The minimum gasket seating pressure on assembly at room temperature. NOTE Only input when a _User defined_ gasket has been selected.                                                                                                                             |
| Pressure Qsmin       | The minimum gasket seating pressure at the design temperature. NOTE Only input when a _User defined_ gasket has been selected.                                                                                                                                   |
| Pressure Qmax        | The maximum gasket surface pressure. NOTE Only input when a _User defined_ gasket has been selected.                                                                                                                                                             |
| Elastic mod EG       | The gasket elastic modulus. NOTE Only input when a _User defined_ gasket has been selected.                                                                                                                                                                      |
| Factor PQR           | For _User defined_ gaskets, from _EN 1591-2:Table 17_ , creep relaxation factor. This is equal to 1.0 if no creep of the gasket is considered.                                                                                                                   |
| Pressure Qi          | From _EN 1591-2:Table 17_ the gasket pressure. NOTE Only input when a _User defined_ gasket has been selected.                                                                                                                                                   |
| Thickness eG         | For _User defined_ gaskets, the initial compressed gasket thickness under contact pressure _Q GO_ given in EN 1591-2 Tables 18 to 37.                                                                                                                            |
