# Base plate data
## Data Input
Description | Enter the base plate description.  
---|---  
Welded to deck | Select _Yes_ if the skirt is welded to a deck, Select _No_ for a conventional base plate calculation.  
Bolt type | Select _ISO Metric series_ or _Inch series_ \- see [Bolting Data](<h9984.htm>).  
Diameter | Select the required bolt size - see [Bolting Data](<h9984.htm>).  
Quantity | Number of bolts.  
Bolt area | Actual cross sectional area of one bolt at the section of least diameter under load, for _User defined_ bolting.  
Minimum bolt pitch | Minimum recommended bolt pitch, for _User defined_ bolting.  
Bolt Material | Bolt material selection from the [Material Database](<h9996.htm>).  
Base Material | Base material selection from the [Material Database](<h9996.htm>).  
## Additional Information
A base plate can only be added to a vessel that has been specified as _Vertical_ in [Item orientation](<h9303.htm>).
## Error Messages
See also [General component error messages](<h9100.htm#1000>).
Conflicting diameters | The _Base plate OD_ must be greater than the adjacent _Skirt OD_ and the _Base plate ID_ must be less than the adjacent _Skirt ID_.
