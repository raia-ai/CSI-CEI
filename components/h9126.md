# Bend data

## Data Input

| Description    | Enter the bend description.                                                                                                                                                                                                                                                                              |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 90 deg. elbow  | Select _Yess_ to define a 90 deg. elbow, or _No_ for a 45 deg. elbow.                                                                                                                                                                                                                                    |
| Diameter       | Select option for input of bend diameter as _Nominal bore_ , _External_ or _Internal_. For standard pipe sizes to ANSI B36.10 or BS 1600 select _Nominal bore_. For other diameters and/or thicknesses select _External_ or _Internal_ - see [\[5\]](../h9998.htm#0005) and [\[11\]](../h9998.htm#0011). |
| Tolerance      | Select option for input of thickness tolerance as either _Absolute_ or _Percentage_.                                                                                                                                                                                                                     |
| Thinning       | Thinning allowance to cater for mill tolerance on bend material. This tolerance will be deducted from the new thickness.                                                                                                                                                                                 |
| Insulation thk | Thickness of any external insulation.                                                                                                                                                                                                                                                                    |
| Material       | Bend material selection from the [Material Database](../h9996.htm).                                                                                                                                                                                                                                      |
