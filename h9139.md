# Vessel wizard data
## Data Input
Units | Select the required units.  
---|---  
Head | Select the required head type - see [Shells & Heads](<h0098.htm#0100>). |   
Diameter: OD-ID-NB | Select option for input of diameter as _Nominal bore_ , _External_ or _Internal_. For standard pipe sizes to ANSI B36.10 or BS 1600 select _Nominal bore_. For other diameters and/or thicknesses select _External_ or _Internal_ \- see [\[5\]](<h9998.htm#0005>) and [\[11\]](<h9998.htm#0011>).  
Material | Shell, head and nozzle material selection from the [Material Database](<h9996.htm>). These can be changed when the vessel wizard has finished.  
Nozzle data | A heading to indicate that the following data applies to the nozzle.  
Nominal bore | Select the required nominal bore for the nozzle - see [Nominal Pipe Sizes & Schedules](<h9986.htm>). NOTE - the software will automatically attach a Flange to this nozzle which corresponds to this diameter.  
Schedule | Select the required pipe schedule - see [Nominal Pipe Sizes & Schedules](<h9986.htm>).  
Design data | A heading to indicate that the following data applies to the Design data.  
Code | Select the required design code from _PD5500_ ,_ASME VIII Div.1_ ,_ASME VIII Div.2_ or _EN13445_.**NOTE: This can be changed at any time when in the software.**  
Design pressure | Design internal pressure acting on the vessel, including the effects of liquid static head and/or pressure drops as applicable.  
Design temp | Vessel design temperature.  
Corrosion: int | Internal corrosion allowance to be deducted from the as built vessel thickness.  
Corrosion: ext | External corrosion allowance to be deducted from the as built vessel thickness.  
See also [Vessel wizard](<h9800.htm#0700>)
