# Tool menu commands
* * *
## Autosave input
From the Tools menu select **Autosave input**. This will toggle the AutoSave option ON or OFF. When it is toggled ON the menu item will be checked, and then when any calculation is performed the software will automatically save the corresponding input data, using the same description as the calculation name. This file can then be read from the Finglow 2D application if required.
* * *
## Change units
From the Tools menu select **Change units** or from the Toolbar click on the button ![](..\\img\\unit.png) to change units and/or convert data - see [Unit Conversion](<h9997.htm#0101>)
* * *
## EFP Editor
From the Tools menu select **EFP Editor**.
This displays the [EFP Editor](<h9703.htm>) dialog box.
* * *
## EFP Linker
From the Tools menu select **EFP Linker**.
This displays the [EFP Linker](<h9704.htm>) dialog box.
* * *
## Highlight output data
From the Tools menu select **Highlight output data**. This will toggle the Highlight output data option ON or OFF. When it is toggled ON the menu item will be checked and any output data that is displayed in red on the screen will be circled on the printed output, as shown below.
![](..\\img\\pblink.png)
* * *
## NozzlePRO settings
From the Tools menu select **NozzlePRO settings**.
This will allow for the input of the pathname to the required NozzlePRO installation - see [NozzlePRO pathname](<h8105.htm#0100>).
* * *
## Reset factory settings
From the Tools menu select **Reset factory settings**.
This will reset the screen size and position back to the default settings. Also the component colours will be reset back the their default settings.
