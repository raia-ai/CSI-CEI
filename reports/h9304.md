# Finglow Model 3D view

From the Toolbar press the ![](https://github.com/raia-ai/CSI-CEI/blob/img/3ddraw.png) button, this will display the current item as a 3D model in a separate window. Below is a list of actions that can be selected from the Toolbar displayed in this window.

| **Button**                                                         | **Action**                                                                                                                      |
| ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------- |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/updraw.png)        | Move the model upwards.                                                                                                         |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/dndraw.png)        | Move the model downwards.                                                                                                       |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/leftdraw.png)      | Move the model to the left.                                                                                                     |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/rightdraw.png)     | Move the model to the right.                                                                                                    |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/restoralldraw.png) | Restore the model to its original size and position.                                                                            |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/rotx1draw.png)     | Rotate the model anti-clockwise about the X-axis. Alternatively move the mouse up while holding the left button down.           |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/rotx2draw.png)     | Rotate the model clockwise about the X-axis. Alternatively move the mouse down while holding the left button down.              |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/roty1draw.png)     | Rotate the model anti-clockwise about the Y-axis. Alternatively move the mouse right while holding the left button down.        |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/roty2draw.png)     | Rotate the model clockwise about the Y-axis. Alternatively move the mouse right while holding the left button down.             |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/rotz1draw.png)     | Rotate the model clockwise about the Z-axis.                                                                                    |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/rotz2draw.png)     | Rotate the model anti-clockwise about the Z-axis.                                                                               |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/restordraw.png)    | Restore the model to its original position in the window.                                                                       |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/view1draw.png)     | View the model along the X-axis in the positive direction.                                                                      |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/view2draw.png)     | View the model along the X-axis in the negative direction.                                                                      |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/zoomindraw.png)    | Zoom in on the model. Alternatively turn the mouse wheel upwards.                                                               |
| ![](https://github.com/raia-ai/CSI-CEI/blob/img/zoomoutdraw.png)   | Zoom out from the model. Alternatively turn the mouse wheel downwards.                                                          |
| Mouse - right click                                                | This will display the [Component Action](../h9000.htm#0001) menu for the component under the cursor.                            |
| Mouse - double click                                               | This will display the component editing screen for the component under the cursor - see [Editing component data](../h9100.htm). |
