# Material manager
This function allows the user to change the material for some or all components in one action.
After selecting _Material manager_ from the _Tools_ menu, the list of materials for the entire model will be displayed.
![](..\\img\\3dmanager.png)
To change the material of a component, click in the check box next to that component.
![](..\\img\\3dmanagerchk.png)
In the _Substitute material_ group box at the bottom left of the screen, press the material selection button. This will display the [Material Database](<h9996.htm>) dialog box, from this select the new material and press _Ok_.
Press the _Change_ button to action the changes selected.
