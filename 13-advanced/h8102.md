## Calculate
From the Tools menu select **Calculate** or from the Toolbar click on the button ![](..\\img\\calc.png)
After first checking the completeness of the design data entered, this option performs the relevant calculations. The Code design requirements and limitations are checked and a message relating to the outcome of the calculations and checks will be displayed in the Calculation Status dialog box which will always be displayed whenever a calculation is performed.
* * *
## Calculation Status
From the Tools menu select **Status** or from the Toolbar click on the button ![](..\\img\\calcstat.png)
This dialog box is displayed every time a calculation is performed and will stay on the screen until the user removes it, or changes any input data. This box will display any system messages generated by the software. To go straight to the relevant section of the help, double click on any message displayed in the list. 
The messages fall into two categories as described in [System Messages.](<h9997.htm#0500>)
Any pages that contain items of data that relate to the issued warnings will have a ![](..\\img\\calc_wrn.png) or ![](..\\img\\calc_bad.png) displayed next to them, depending on the warning type.
